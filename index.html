Webpad

This is just a simple web browser text editor i made.

So basically you type into that box that says "Type here..." than hit "Write", you will write text onto the note.

You can download your text by clicking the button that says "Download Note"

You can save your note to the browser's localStorage by clicking the "Save to Browser" button and to load the note from there click "Load Note from Browser" to load from the browser's localStorage'

Note that loading from localStorage will load the latest saved note.

Webpad for Desktop will release soon! basically its just Webpad as a desktop app via Electron
